# 🚀 Ardio – Digital Marketing Agency Landing Page

A modern, clean, and responsive landing page for a digital marketing agency. Built using HTML, CSS, and Tailwind CSS, this project focuses on minimal design, usability, and a smooth user experience.

---
## 📸 [Live Preview](https://amitchandradas2004.github.io/Agency-landing-page/)

---

## ✨ Features

- Clean and minimalist UI design  
- Fully responsive layout (mobile-friendly)  
- Built with Tailwind CSS (utility-first approach)  
- Well-structured sections:
  - Hero Section  
  - Experience Overview  
  - Statistics Counter  
  - Process Section  
  - Portfolio Showcase  
  - Newsletter Subscription  
- Modern typography and color scheme  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Tailwind CSS
